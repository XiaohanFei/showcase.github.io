About the Project

This project is a Yelp-like application designed for users to find, review, and add local places. It features a React frontend, a Node.js and Express.js backend, and a PostgreSQL database hosted on Heroku.

Due to an issue with Heroku, the live instance of this application is currently down. However, we are working on recovering it as soon as possible. In the meantime, please feel free to explore the project through these screenshots and the GitHub repository.
